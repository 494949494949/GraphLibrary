# Техническое задание
**Вариант 2.** Библиотека для работы с графами (ориентированными и
неориентированными). Реализация генератора случайных графов, реализация
алгоритмов работы с графами.
## Требования
- Отдельный демо-проект: собрать в .jar файл через мавен (проще), импортировать её как библиотеку в другой проект через настройку либы. Можно как мавен-зависимость импортировать (локальный репозиторий).
- Библиотека должна быть покрыта тестами.
- Можно реализовать как список смежности (Map)
- Реализация добавления веса и прочие нюансы интерфейса висят на мне
- В результате должен быть граф (интерфейс и его реализации), добавление вершин и рёбер, алгоритмы поиска в глубину и ширину, алгоритм Дейкстры, поиск компонент связности.
- Орграфу можно унаследоваться
- Можно спереть графический интерфейс из другого проекта
# Реализация
Приложены два проекта: **GraphLib** и **GraphSamples**.
## GraphLib
Это **моя** реализация самой библиотеки, покрытая тестами и реализующая техническое задание.
Демо-проектом де-факто выступает **GraphSamples**, но также в **GraphLib** есть директория *com.greefon.graphs.test.demo*, где находится класс, в котором я делал минимальные проверки работоспособности исходной библиотеки.
## GraphSamples
Проект **Д.И.Соломатина**, модифицированный для визуализации графов из моей библиотеки через адаптеры.
Библиотека компилируется через Maven (`mvn package`), после чего .jar файл подключается через Intellij IDEA без особых трудностей.
